# AI Vibe Workflow - 브랜딩 정리

## 📌 핵심 메시지

### 서비스 이름
**AI Vibe Workflow**

### 서비스 설명 (한줄)
**AI 개발자를 위한 3단계 워크플로우 자동화 플랫폼**

### 상세 설명
반복적인 설정 작업에서 벗어나 핵심 프롬프트 개발에 집중하세요.  
Intro(초기화) → Contents(개발) → Deploy(배포)의 3단계로 AI 앱 개발을 자동화합니다.

## 🚀 3단계 워크플로우

### 1️⃣ Intro (프로젝트 초기화)
**목적**: 기존 자산 재활용으로 프로젝트 설정 자동화

**주요 기능**:
- GitHub 저장소 자동 생성
- 프롬프트 재사용 및 템플릿 적용
- 초기 설정 시간 대폭 단축 (30분 → 5분)

**색상**: 🔵 파란색 계열 (신뢰, 시작)

### 2️⃣ Contents (프롬프트 개발)
**목적**: AI 프롬프트 개발과 테스트에 집중

**주요 기능**:
- 멀티 AI 모델 비교 (GPT, Claude, Gemini)
- 고급 프롬프트 에디터 (Monaco Editor)
- 실시간 결과 비교 및 분석

**색상**: 🟢 초록색 계열 (성장, 창작)

### 3️⃣ Deploy (배포 자동화)
**목적**: GitHub, 데이터베이스, 클라우드 배포 자동화

**주요 기능**:
- Vercel/Netlify 자동 배포
- Supabase 데이터베이스 설정
- 환경 변수 자동 구성 (45분 → 7분)

**색상**: 🟠 주황색 계열 (에너지, 완성)

## 💡 핵심 가치 제안

### 3가지 핵심 KPI
1. ⚡ **설정 시간 60% 단축**: 75분 → 30분
2. ♻️ **자산 재사용률 70% 달성**: 7/10 프로젝트에서 기존 자산 활용
3. 💰 **AI 크레딧 30% 절감**: 월 $100 → $70

### 타겟 사용자
- AI 프롬프트 엔지니어
- AI 앱 개발자
- 스타트업 CTO/개발팀 리더
- 프리랜서 개발자
